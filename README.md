# spacexOS

Retro web operating system made with plain HTML, CSS, and JavaScript for Hack Club.

## Features

- Multiple windows which can be dragged
- 7 apps: About, Notes, Terminal, Synth, Draw, Calculator, Game
- Space backgrounds that can be changed (Wallpaper button)
- Over 20 terminal commands
- Meteor Dodge game
- Shortcuts (1-7 to launch apps)

## To run

```bash
cd /path/to/spacexOS
python3 -m http.server 8000
Then go to http://localhost:8000 on your browser.

Apps
About — information about me and my skills
Notes — save your notes into localStorage
Terminal — over 20 commands (joke, roll, flip, 8ball, etc.)
